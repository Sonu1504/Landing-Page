# Landing Page

## Overview
A visually engaging fan-inspired tribute website celebrating **13 Years of One Direction**. The project is built using HTML and CSS and features a full-screen background image, navigation menu, promotional content, social media/music platform icons, and a call-to-action button.

## Features
- Full-screen responsive background image
- Navigation menu with hover effects
- Custom typography and branding
- One Direction anniversary-themed landing page
- Call-to-action ("LET'S GO") button
- Music platform icons (Spotify, Apple Music, SoundCloud, Deezer)
- Footer with copyright and policy information
- Interactive hover animations
